Monitor and alert program
Monitor the component process and alert with SMS when it go dead.
#Dependency
*	Linux
*	python 2.6
*	SMS python API
#Install
./install.sh

#Config
*	conf/config
*	phone numbers
*	services
*	mailServer

#Start
./monitor start

#Stop
./monitor stop
